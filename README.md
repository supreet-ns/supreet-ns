# IoT Soil Moisture Detector 🌿

A first-year engineering project. This device monitors soil health in real-time using an ESP32 and sends data to a mobile app.

## 🛠️ Components
- **Microcontroller:** ESP32
- **Sensor:** Capacitive Soil Moisture Sensor v1.2
- **Connectivity:** Wi-Fi (HTTP/MQTT)

## 🚀 Features
- Real-time moisture level tracking.
- Low-power mode for battery efficiency.
- Mobile notification when the plant needs water.

## 📂 Project Structure
- `/src`: Source code (C++/Arduino)
